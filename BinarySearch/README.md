# pseudo-code
1. 현재 배열이 오름차순인지 확인한다. 
    1. 현재 배열의 가장 작은 인덱스가 가장 큰 인덱스보다 크거나 같을 때 까지 반복한다. 
        1. 찾으려는 값이 중간 값과 같은지 확인한다. 
            1. 같다면, 중간 값을 반환한다. 
        2. 찾으려는 값이 중간 값보다 큰지 확인한다. 
            1. 크면, low 값을 중간 값의 다음 위치의 값으로 변경한다. 
        3. 작으면, high 값을 중간 값의 이전 위치의 값으로 변경한다.
